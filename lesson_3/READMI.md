## Toán tử gán (Assigment):

          Operator            Exam                 The same
->           =              x = y                x = y
->          +=              x += y               x = x + y
->          -=              x -= y               x = x - y
->          *=              x *= y               x = x * y
->          /=              x /= y               x = x / y
->          **=             x **= y              x = x ** y
